Recent Projects

ELITE GMART

EliteGmart is an e-commerce platform specializing in Pooja Samagri (worship materials) and related products, allowing users to conveniently purchase items based on their specific requirements.

Mail OTP Validation was implemented for user account verification. This module was developed using both Nodemailer and an SMTP server.

Server-Side and Client-Side Validations were implemented for the user registration and login modules, as well as for adding, removing, and updating products with their cost prices.

Challenges: The verification emails for user account verification were not consistently reaching users. To resolve this, I implemented Nodemailer with an SMTP server, ensuring efficient and reliable email delivery.

ONLINE EXAM PORTAL

Worked on Admin Portal of Online Exam Portal, where the admin can create the exam paper with specific requirements and can give access to the respective students to start the test with robust validations.

Used Springboot for the backend processing. Used Cache storage for data storage and data retrieval as well. Used MySQL for backend database.

Challenges: The data was not being accurately stored or retrieved from Cache Storage. To resolve this, I implemented separate functionalities for each field to ensure proper data handling. Leveraging these functionalities, I developed a preview page that allows users to review their stored data efficiently.

kryssolutions

Kryssolutions.com is a project that highlights the services offered by Kryssolutions.

I implemented the backend for the contact form using Nodemailer along with SMTP server to send form submissions directly to the company's email. I integrated Multer to handle form-data as middleware and utilized MySQL to store the submitted data in the database.

Challenges and Solutions: During the project, I encountered an issue where the server was not receiving form submissions directly due to the involvement of a third-party website in the process. To address this, I eliminated the third-party dependency and integrated Nodemailer, a Node.js module, along with an SMTP server. This solution ensured that the form data was successfully stored in the MySQL database and delivered directly to the company's email.
